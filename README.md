# Meal Inspiration

A simple web application that provides random daily meal suggestions.


## Files

- `index.html` - Main web interface
- `meal-plans.json` - all the meal plans
- `README.md` - This file


## Principles

All meals follow these principles:
- Each lunch/dinner includes: grain + legume/protein + augmenting vegetable + extractive vegetable + oils + spices
- Breakfasts are porridge or sauteed fruit
- No aliums or night shades (garlic, onion, bell pepper, tomato, eggplant, potato)
- Variety of grains, legumes, and seasonal vegetables
